# angular-md directive demo This directive renders [Markdown](http://daringfireball.net/projects/markdown/) text. It's built on blazingly fast markdown parser [marked](https://github.com/chjj/marked). - This is a bullet point 1. - This is a bullet point 2. - This is a bullet point 3. Code block is supported, too: ```js for (var i = 0; i < 10; i++) { console.log('angular-md is cool!'); } ```
